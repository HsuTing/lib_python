```
- fromweb(url, name)

	input two array. one is url, and another is name.
	it will get all html from your url,
	and your name will be the name of output file.

	example:

		url = [	'http://opendata.epa.gov.tw/ws/Data/SGWC/?$format=xml', ]
		name = [ 'SGWC' ]
		fromweb(url, name)
```
